<Note>Stub — overview of the four packages and how they compose.</Note>

| Package | Purpose |
|---|---|
| [`@afauthhq/core`](/sdk/typescript/core/overview) | Shared primitives: `did:key`, RFC 9421, recipient types, error envelope. |
| [`@afauthhq/agent`](/sdk/typescript/agent/overview) | Client side: `Agent.generate()`, `signRequest`, protocol builders. |
| [`@afauthhq/server`](/sdk/typescript/server/overview) | Service side: `Verifier`, `Server`, resolvers, rate limiter, attestor. |
| [`@afauthhq/worker`](/sdk/typescript/worker/overview) | Cloudflare Workers bindings: `createWorker`, KV/D1 stores. |
